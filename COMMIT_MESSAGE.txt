Modernize project and add real-time flight tracking capabilities

Major enhancements to the flights-analysis project:

## Modernization
- Updated Python code to 3.10+ with type hints, pathlib, f-strings
- Replaced deprecated Basemap with Cartopy for map projections
- Added modern project configuration (pyproject.toml, requirements.txt)
- Implemented CLI interfaces with argparse for both visualization scripts
- Added YAML configuration file support for customization
- Created comprehensive documentation and examples

## Real-Time Flight Data Integration
- NEW: fetch_flights.py - Fetch live flight data from ADS-B Exchange
- Geographic filtering by bounding box or radius
- Route aggregation with configurable thresholds
- Compatible CSV output format with existing visualization tools

## Specific Flight Tracking
- NEW: track_flight_opensky.py - Track specific flights by callsign (FREE)
  - Uses OpenSky Network API (no key required)
  - Live position updates with detailed telemetry
  - Save tracks to CSV with full flight parameters
  - Example: track_flight_opensky.py UAL2212 --follow --updates 20
  
- NEW: track_flight.py - Alternative using ADS-B Exchange (requires API key)

## Flight Track Visualization
- NEW: visualize_track_simple.py - Comprehensive flight analysis plots
  - 6-panel visualization: path, altitude, speed, climb rate, statistics
  - Generates publication-quality graphics
  - Works with any tracked flight CSV data
